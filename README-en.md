## Setup
1. **Clone the repo**
```
$ git clone https://github.com/ZDfordream/FlutterTianYue.git
$ cd FlutterTianYue
$ flutter packages get
```
2. **Running:**
```
$ flutter run
$ Tip:if build fail,just build again.
$ flutter version : 1.0.0
```
